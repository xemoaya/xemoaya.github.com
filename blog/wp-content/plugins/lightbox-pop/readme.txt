Lightbox Pop is handy WordPress plugin which allows you to create simple, non-annoying lightbox popups in your blog.

== Description ==

This plugin is used to create a lightbox with custom content in your site. You can configure lightbox position settings (height, width ,top, left) in percentage or pixels, display logic related settings (time delay after page load, number of pages to browse for lightbox to appear, lightbox repeat interval for a user) and style settings(z-index, overlay opacity, color, border etc). You can use the lightbox to display any type of contents such as special promotions, subscription forms, social media icons, video presentations and much more.

= Features =

* Full control on lightbox content
* Standard WordPress content editor for editing content of lightbox
* Timer based display
* Display based on browsed number of pages
* Lightbox position settings
* Lightbox dimension control
* Lightbox color settings
* Advanced css settings

== Installation ==

1. Extract `lightbox-pop.zip` to your `/wp-content/plugins/` directory.
2. In the admin panel under plugins activate Lightbox Pop.
3. You can configure the settings from Lightbox-Pop menu.
4. It should now be completely set up and functional.

If you need any further help, you may contact our [support desk](http://xyzscripts.com/members/index.php?page=user/support "XYZScripts Support").

== Frequently Asked Questions ==

= 1. The Lightbox Pop is not working properly =

Please check the wordpress version you are using. Make sure it meets the minimum version recommended by us. Make sure all files of the `lightbox pop` plugin uploaded to the folder `wp-content/plugins/`

= 2. Can I use this plugin with other lightbox plugins ? =

It may interfere with each other if there are 2 or more lightbox plugins. 

= 3. Can I add my own styles for the overlay ? =

Yes, you can change the style of the overlay from the settings page. However you cannot use an image for the overlay as of now.

= 4. Can I set lightbox repeat interval in minutes ? =

The lightbox repeat interval must be configured in hours. So if you want lightbox to be repeated say, every 15 minutes, you must set the repeat interval as 15/60 , ie,  0.25

= 5. What kind of contents can be placed in lightbox ? =

You may  use the lightbox to display any type of contents such as special promotions, subscription forms, social media icons, video presentations and much more.

= 6. Can I display lightboxes in specific pages only ? =

Yes, you can do this using the premium version of our lightbox which can be purchased from our site.

= 7. Can I create multiple lightboxes for different pages ? =

Yes, you need to purchase the premium version for this feature.
